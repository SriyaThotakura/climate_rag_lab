01

What is this?

Climate RAG Lab is a hands-on classroom tool for building and evaluating retrieval-augmented generation pipelines using open-source models. Students wire up a local Streamlit app, compare prompt strategies, and measure retrieval quality — all on their own laptops with Docker + Ollama.

Prompt Styles

Three Modes

Zero-shot, few-shot, and evidence-first chain-of-thought prompting for direct comparison.

Evaluation

Built-in Harness

Retrieval hit rate, lexical groundedness, keyword coverage, and optional LLM-as-judge scoring.

Local-First

No API Keys

Runs entirely on Ollama + ChromaDB. No cloud dependencies, no cost per query.
